🔗 連接 GitHub Gist

帳單資料會儲存在你 GitHub 帳號下的一個 private Gist。需要一個 Personal Access Token（PAT）。

建立新 Gist
使用既有 Gist
1. 點下面按鈕，會打開 GitHub Token 產生頁（已預填正確 scope）
2. 確認 gist 已勾選 → 拉到底點「Generate token」
3. 複製出現的 token（只會顯示這一次！）→ 貼到下方欄位
1. 你的 Gist ID 可以從原本裝置複製：開設定面板 → 顯示 Gist ID
2. 同樣需要一個有 gist 權限的 token（可重用之前的，或開新的）
ℹ️ Token 與 Gist ID 會存在這個瀏覽器的 localStorage，帳單資料本身只存在 GitHub Gist。
